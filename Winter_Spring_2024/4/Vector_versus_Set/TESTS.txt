Добавление элемента в вектор выполняется за О(1), значит добавление N элементов займет О(N).
Вставка элемента в set выполняется за О(log(N)), значит добавление N элементов займет O(Nlog(N)), поэтому вставка происходит дольше.

Для сортировки std::sort худшее время сортировки составляет O(Nlog(N)), поэтому сортировка вектора происходит медленне, чем добавление.
